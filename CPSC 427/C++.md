## Templates
- if we want to create many similar classes that use different data types
```

class Vec2
{
public:
	T x;
	T y;
};
```

## Memory Allocation
```
void func() {
	int val = 10;           // stack
	int *pVal = new int;    // heap
	val += 1;
	*pVal = 7;              // dereference pointer
	*pVal += 1;
	delete pVal;            // free memory
}
```
